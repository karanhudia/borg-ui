==================================
borgui.borg_ui Release Notes
==================================

.. contents:: Topics

v1.0.0
======

Release Summary
---------------

Initial release of the ``borgui.borg_ui`` Ansible collection.

New Modules
-----------

- ``borgui.borg_ui.borg_ui_repository`` - Manage borg-ui backup repositories
- ``borgui.borg_ui.borg_ui_schedule`` - Manage borg-ui scheduled backup jobs
- ``borgui.borg_ui.borg_ui_connection`` - Manage borg-ui SSH connections
- ``borgui.borg_ui.borg_ui_notification`` - Manage borg-ui notification channels
- ``borgui.borg_ui.borg_ui_backup`` - Trigger, monitor, and cancel backup runs
